# Boatrace Open API for Results

ボートレース（競艇）の結果データを取得できるWeb APIです。
GitHub Pagesを使用して静的なJSONファイルとして配信しています。

## エンドポイント
```
https://boatraceopenapi.github.io/results/v2/{日付}.json
```

## サンプル
- [https://boatraceopenapi.github.io/results/v2/20250715.json](https://boatraceopenapi.github.io/results/v2/20250715.json)
- [https://boatraceopenapi.github.io/results/v2/today.json](https://boatraceopenapi.github.io/results/v2/today.json)

## 関連
| 対象 | リポジトリ | エンドポイント |
|:---|:---|:---|
| 出走表 | [Boatrace Open API for Programs](https://github.com/BoatraceOpenAPI/programs) | https://boatraceopenapi.github.io/programs/v2/{日付}.json |
| 直前情報 | [Boatrace Open API for Previews](https://github.com/BoatraceOpenAPI/previews) | https://boatraceopenapi.github.io/previews/v2/{日付}.json |
